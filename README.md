## 🚀 KHỞI ĐỘNG NHANH

### 1. Khởi động MySQL

```bash
.\START_MYSQL.bat
```

### 2. Khởi động Backend

```bash
.\START_API.bat
```

### 3. Khởi động Frontend

```bash
cd store-management-frontend
pnpm install
pnpm run dev
```

### 4. Truy cập

- **Frontend:** http://localhost:3000
- **Backend API:** http://localhost:5122/swagger

### 5. Đăng nhập

- **Username:** `admin`
- **Password:** `123456`
